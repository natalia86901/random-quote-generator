Random Quote Generator

A simple and lightweight browser-based app that displays a random motivational or reflective quote each time a button is clicked.

Features

- Click a button to reveal a random quote
- Minimal vanilla JavaScript — no frameworks or dependencies

Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

How to Use

1. Clone the repository:
   git clone https://github.com/your-username/random-quote-generator.git
2. Open `index.html` in your browser.
3. Click the button to generate a quote.
